About CivicPulse

Citizen Grievance Intelligence System — Hackathon 2024 Submission

Transforming Governance with Data Intelligence

CivicPulse is a full-stack Django + MongoDB analytics platform that processes citizen grievance data to surface systemic public service failures, benchmark departmental performance, and generate AI-driven recommendations for governance improvement.

🐍 Python 3.11 🦄 Django 4.2 🍃 MongoDB 🐼 Pandas 🔢 NumPy 📊 Matplotlib 📈 Chart.js 🎨 CSS3

🧩 System Modules

📥 Data Ingestion

Reads grievance CSV using pandas with validation, missing value handling, and format checks.

🧹 Data Cleaning

Deduplication, null handling, column normalization, date parsing, and derived field generation.

📊 EDA Engine

Pandas + NumPy-powered analysis: KPIs, trends, dept/region/category breakdowns, efficiency metrics.

🔤 Text Analysis

Keyword frequency extraction, grievance theme clustering, and citizen dissatisfaction indicators.

💡 Smart Insights

Auto-generated alerts, efficiency scores, risk indicators, and actionable recommendations engine.

📈 Chart Service

Matplotlib chart generation: trend, bar, pie, comparison, growth rate, severity distribution.

🍃 MongoDB Layer

Stores cleaned records, insights, alerts, report history, and dashboard cache with graceful fallback.

🖥️ Django Views

10 view functions + 4 REST API endpoints with filter support and JSON responses.

🎨 UI Layer

Premium dark dashboard with sidebar, KPI cards, interactive charts, search, and filters.

🔄 System Workflow
1
CSV Ingestion — GrievanceEngine reads data/grievance_data.csv using pandas.read_csv() with file existence validation
2
Data Cleaning — Deduplication, null filling, date parsing, column normalization, and derived fields like resolution_days
3
Analysis — Pandas groupby aggregations compute department/region/category metrics, trends, and efficiency scores
4
MongoDB Storage — Cleaned records, insights, and alerts stored in MongoDB collections with graceful SQLite fallback
5
Chart Generation — Matplotlib generates 8 chart types saved as PNG files in /static/charts/
6
Django Views — Views compile analysis data, chart URLs, and insights into template context dictionaries
7
Dashboard Rendering — Jinja2 templates with Chart.js render interactive KPI cards, charts, tables, and recommendations
